Skybadge

Control a skytech 153.115 moving message badge under linux

Download as .zip Download as .tar.gz View on GitHub

Pour compiler, installer ou desinstaller skybadge, utilisez les traditionnels, make, make install et make uninstall.

Organisation du répertoire :

A la racine : les sources de skybadge. Dans le répertoire man : le source de la page de manuel. Dans le répertoire ttyStest : un module kernel qui permet de simuler un port serie (utile pour developper sur un portable non equipe de port serie).
